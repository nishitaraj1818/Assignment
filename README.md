# SQL-Based Sales Data Analysis

## Objective
Analyze the Superstore Sales dataset using SQL to uncover business insights.

## Tools Used
- MySQL / PostgreSQL / SQLite
- SQL
- Jupyter Notebook
- Python (Pandas)
- Git & GitHub

## Dataset
Use the **Superstore** dataset (`sales_data.csv`).

## SQL Concepts Covered
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- LIMIT
- SUM, AVG, COUNT
- HAVING
- Duplicate Detection
- NULL Value Checks

## Business Questions Solved
1. Total sales by category
2. Top 10 selling products
3. Monthly sales trend
4. Top 10 customers
5. Region-wise sales
6. Duplicate record detection
7. Data quality validation

## How to Run
1. Import `sales_data.csv`.
2. Execute `sql_analysis.sql`.
3. Review results.
